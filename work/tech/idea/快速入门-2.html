# 可能出现的问题

# jetty 启动时可能会报错

[WARNING] Problem processing jar entry sun/applet/AppletPanel.class

由于 JDK 版本问题导致类似错误，但是不影响项目启动完成。

POM.xml，文件中配置的 jetty plugin 版本为父级 POM 中的版本

<parent>
    <groupId>cn.com.yitong</groupId>
    <artifactId>yitong-parent</artifactId>
    <version>4.0.0-MYBATIS</version>
</parent>

<jetty.version>8.1.10.v20130312</jetty.version>

4.0.0-MYBATIS ：8.1.10.v20130312

突然发现，项目启动使用 1.8 版本 JDK

修改为 1.6 之后，Idea 提示

发现我本地配置 maven 版本为 3.6.x，切换 maven 版本使用 apache-maven-3.0.5 得以解决。
